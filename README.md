Run migration (optional, as sqlite.db is included):\n    ```bash\n    npx drizzle-kit push\n    ```\n3.  Start the development server:\n    ```bash\n    npm run dev\n    ```\n4.  Open [http://localhost:3000](http://localhost:3000)\n\n## Admin Access\n\n*   URL: `/admin`\n*   Default Password: `password123` (configured in `.env`)\n
